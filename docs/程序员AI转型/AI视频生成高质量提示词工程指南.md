# AI视频生成高质量提示词工程指南

> 好的提示词 = 镜头语言 + 主体描述 + 光影氛围 + 运动轨迹 + 情感基调
> 这份文档持续更新，每次生成视频前先来这里找灵感

---

## 一、核心公式

```
[镜头类型] + [主体] + [动作/状态] + [环境] + [光影/色调] + [运动轨迹] + [氛围/情感]
```

**示例拆解：**
```
中景跟随 → 一只橘猫 → 从左向右踱步 → 午后阳光洒落的木地板 → 暖黄色侧光 → 缓慢横移 → 慵懒治愈
```

---

## 二、镜头类型速查表

| 缩写 | 镜头 | 用法 |
|------|------|------|
| WS | Wide Shot 远景 | 交代大场景 |
| MS | Medium Shot 中景 | 人物/动物主体 |
| CU | Close-Up 特写 | 表情、细节 |
| LS | Long Shot 全景 | 环境+人物 |
| POV | 主观镜头 | 观众代入感 |
| 低视角 | Worm's Eye | 宏大感 |
| 航拍 | Top View | 全局展示 |

## 三、抖音爆款提示词模板

### 模板1：治愈系小动物
```
[主体描述]：一只橘色短毛猫，毛发光泽微亮
[镜头]：中景跟随，镜头从猫前方缓慢推进
[动作]：猫咪回头眨眼，然后继续向前踱步
[环境]：温馨的客厅，暖色调木地板，阳光从窗户斜射进来
[光影]：逆光勾勒毛发边缘光，焦外散景
[色调]：暖黄色为主，营造午后慵懒氛围
[运动]：镜头与猫同步移动，保持1米距离
[情感基调]：治愈、温暖、放松
```

### 模板2：国风古风
```
[主体描述]：穿青色汉服的少女，长发及腰，发丝随风飘动
[镜头]：侧面跟拍，镜头平行移动
[动作]：少女回眸，眼神含笑，步伐轻盈
[环境]：竹林小径，竹叶沙沙作响，远处有古建筑飞檐
[光影]：阳光穿透竹叶形成斑驳光影，人物轮廓光
[色调]：青绿+暖白，国画色调
[运动]：镜头环绕少女180度，配合她的步伐节奏
[情感基调]：清雅、脱俗、诗意
```

### 模板3：赛博朋克/科技感
```
[主体描述]：女性机械义体，银色金属质感，蓝色LED光纹
[镜头]：低视角仰拍，展现身高压迫感
[动作]：机械义体从暗处缓步走出，关节发出微弱蓝光
[环境]：未来城市夜景，霓虹灯光，高楼林立
[光影]：蓝色+紫色霓虹光，雨后地面反光
[色调]：赛博朋克蓝紫，冷峻金属质感
[运动]：镜头缓慢上升，从脚部特写拉到全身
[情感基调]：神秘、力量、科技感
```

### 模板4：美食/产品展示
```
[主体描述]：新鲜牛油果牛排，酱汁缓缓淋下
[镜头]：俯拍45度，固定镜头
[动作]：酱汁从上方淋下，凝固一瞬，特写酱汁在牛排上的流动
[环境]：白色大理石台面，简约西式餐具背景
[光影]：顶光，食物上方柔光箱，反光补阴影
[色调]：暖棕色+奶白色，食欲感
[运动]：酱汁慢动作凝固（0.5倍速效果）
[情感基调]：精致、高级、食欲大开
```

### 模板5：情感故事/剧情
```
[主体描述]：中老年男人，60岁左右，眉头微皱，神情沧桑
[镜头]：面部特写+环境交代交替
[动作]：老人坐在老旧木椅上，双手捧着一张发黄的老照片，沉默良久
[环境]：80年代农村老屋，斑驳的土墙，老式挂钟滴答作响
[光影]：窗外透进的自然光，光束中有灰尘飘浮
[色调]：泛黄色怀旧滤镜，低饱和度
[运动]：镜头缓慢从照片拉远，揭示整个老屋环境
[情感基调]：怀旧、感动、岁月感
```

## 四、运动轨迹描述

| 运动 | 英文描述 | 效果 |
|------|---------|------|
| 推镜头 | Dolly In, Push In | 聚焦、压迫感 |
| 拉镜头 | Dolly Out, Pull Back | 揭示全局 |
| 横移 | Truck Left/Right | 平行跟踪 |
| 环绕 | 360° Orbit | 戏剧张力 |
| 跟随 | Follow Shot | 沉浸感 |
| 升降 | Crane Up/Down | 宏大感 |
| 手持抖动 | Handheld Shake | 真实感、紧张感 |
| 慢动作 | Slow Motion | 戏剧化 |

## 五、光影氛围关键词

### 光线类型
```
逆光 Backlight      → 轮廓光、氛围感
侧光 Side Light     → 戏剧感、立体感
顶光 Top Light      → 恐怖片常用
柔光 Soft Light    → 治愈、人像
硬光 Hard Light    → 力量感、分明阴影
伦布朗光 Rembrandt → 人像经典
```

### 色调关键词
```
暖色调 Warm Tone     → 温馨、怀旧、治愈
冷色调 Cool Tone     → 科幻、孤独、冷静
低饱和 Desaturated  → 电影感、复古
高饱和 Vibrant       → 活力、年轻、电商
黑白对比 B&W High Contrast → 戏剧张力
```

## 七、即梦/可灵专用提示词格式

### 即梦（中文提示词模板）
```
【主体】：描述人物/物体外观
【场景】：环境细节
【镜头】：镜头类型和运动
【光效】：光线和色调
【动作】：具体运动描述
【时长】：5-15秒
```

### 可灵（英文提示词模板）
```
A [主体], [动作], [环境], [光影], shot with [镜头类型], [情感], cinematic
```

## 八、避坑指南

### ❌ 不要这样写
- "生成一个好看的视频"（太模糊）
- "猫在走路"（信息不足）
- "效果好"（AI无法理解）

### ✅ 要这样写
- 具体动作：猫从左侧入画，回头看镜头，3秒后向右离开
- 具体光影：逆光勾勒毛发边缘，侧光打在脸部，焦外虚化背景
- 具体情感：温馨治愈，午后慵懒氛围，色调偏暖黄
