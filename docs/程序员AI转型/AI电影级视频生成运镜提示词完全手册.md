## 一、先搞清楚一个问题：为什么你的AI视频看起来假？

**核心原因：你把AI当搜索引擎用，不是当导演用。**

99%的人写提示词是这样的：
```
"生成一个好看的视频" ❌
"猫在走路" ❌
"一个女人在跳舞" ❌
```

抖音上那些让你觉得"很真实"的视频，创作者是这样写的：
```
"女人在竹林中起舞，侧跟镜头，镜头随她旋转180度，
逆光勾勒发丝轮廓光，背景虚化如梦似幻" ✅
```

**AI听不懂"好看的视频"，但AI听得懂"逆光发丝轮廓光+180度旋转+背景虚化"。**

## 五、A-Roll + B-Roll 混合制作法（专业抖音做法）

这是抖音爆款的核心秘密——**真人配音 + AI画面 = 真实感爆炸**

### 5.1 什么是A-Roll和B-Roll

```
A-Roll（主素材）：直接传达信息的部分
  = 真人对着镜头说话 / 真人出镜表演
  
B-Roll（辅助素材）：增强视觉效果的AI生成片段
  = 配合配音的AI场景画面
```

**高评论视频的制作逻辑**：
```
真人出镜说话（建立信任）
    ↓
配合AI生成的B-Roll画面（增强视觉效果）
    ↓
真人出镜总结（引导互动）
```

### 5.2 AI + 真人制作流程

```
第一步：真人录制A-Roll
  → 对着镜头说内容（手机即可）
  → 录音清晰，光线均匀（面向窗户）

第二步：用AI生成B-Roll
  → 根据配音内容，用即梦生成对应画面
  → 写好运镜提示词（参考本文档的22种运镜）

第三步：用剪映剪辑合成
  → A-Roll放主轨
  → B-Roll放视频轨，音画同步
  → 添加字幕、转场
```

## 六、角色一致性秘诀（让人物在多个片段保持一致）

即梦3.0支持角色锁定，以下是关键操作：

```
【角色描述公式】（越具体越好）：
"亚洲女性，24岁，圆脸，下巴微尖，
左眼下方有泪痣，黑色长发及腰，
身着靛蓝汉服，领口有白色刺绣，
右手腕有红色手绳"
```

**提示词开头固定写**：
```
角色一致性：亚洲女性，24岁，下巴微尖，
左眼下方有泪痣，黑发及腰，靛蓝汉服
```

这样即梦会在整个视频生成过程中保持这个角色描述的一致性。
