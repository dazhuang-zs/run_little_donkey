## 路线A：Java后端工程师（经典稳健）

### A.1 岗位画像

**做什么**
Java后端工程师负责服务器端业务逻辑开发、系统架构设计和性能优化。这是国内岗位最多、生态最成熟的技术方向。

**适合谁**
- 逻辑思维强，喜欢解决复杂问题
- 能接受持续学习（Java生态更新快）
- 追求稳定就业，不想押注新兴方向
- 普通院校学生友好（岗位多，机会多）

**发展路径**
```
初级工程师（1-2年）→ 中级工程师（2-4年）→ 高级工程师（4-6年）→ 架构师/技术专家（6年+）
     8K-15K      →      15K-25K      →      25K-40K      →        40K-70K+
```

**普通院校策略**
- Java岗位多，双非也有机会进中大厂
- 重点积累项目经验，学历短板用技术弥补
- 建议先入行，再跳槽提升

### A.3 学习路径（4个月）

#### 阶段1：Java基础+数据库（4周）

**第1周：Java基础强化**
- 面向对象、集合框架、异常处理
- IO/NIO、反射、注解
- 每天编码4小时，完成50道LeetCode简单题

**第2周：多线程与并发**
- 线程基础、线程池、锁机制
- JUC包：CountDownLatch、CyclicBarrier、Semaphore
- 完成一个多线程下载器项目

**第3周：JVM基础**
- 内存模型、类加载机制
- GC算法、垃圾收集器
- 使用VisualVM进行内存分析

**第4周：MySQL数据库**
- SQL语法、索引设计
- 事务、锁、MVCC
- 完成一个图书管理系统（JDBC版）

**阶段产出**：
- 掌握Java核心知识
- 能独立编写CRUD程序
- 理解数据库基本原理

---

#### 阶段2：SpringBoot+项目实战（6周）

**第5-6周：SpringBoot基础**
- Spring IOC、AOP原理
- SpringBoot自动配置、Starter开发
- SpringMVC、RESTful API设计
- 完成用户管理系统（单体版）

**第7-8周：MyBatis+Redis**
- MyBatis原理、Mapper开发
- Redis数据类型、缓存设计
- 集成SpringBoot Cache
- 优化用户管理系统（加缓存）

**第9-10周：项目实战-电商系统**
- 需求分析、数据库设计
- 用户/商品/订单模块开发
- 购物车、订单流程实现
- 使用Redis做会话管理

**阶段产出**：
- 完整的电商系统单体版
- 掌握SpringBoot开发流程
- 理解缓存设计思想

---

#### 阶段3：分布式+微服务（4周）

**第11周：Spring Cloud基础**
- 服务注册发现（Nacos）
- 负载均衡（Ribbon）
- 服务调用（OpenFeign）
- 将电商系统拆分为微服务

**第12周：消息队列**
- Kafka/RabbitMQ基础
- 异步处理、削峰填谷
- 订单超时取消（延迟队列）

**第13周：分布式技术**
- 分布式锁（Redisson）
- 分布式事务（Seata）
- 幂等性设计

**第14周：性能优化**
- SQL优化、索引优化
- JVM调优实战
- 压测与性能分析

**阶段产出**：
- 微服务版电商系统
- 掌握分布式核心概念
- 具备性能优化能力

---

#### 阶段4：面试准备+简历优化（2周）

**第15周：八股文整理**
- Java基础、并发、JVM
- Spring、MySQL、Redis
- 分布式、微服务
- 整理成自己的面试题文档

**第16周：简历+面试**
- 简历撰写（突出项目亮点）
- 模拟面试（自我介绍+项目讲解）
- LeetCode中等题刷题（每天3道）
- 开始投递简历

### A.6 求职策略

#### 简历重点
1. **项目经历**：写2-3个核心项目，突出技术难点
2. **技术栈**：按熟练程度分层列出
3. **开源贡献**：GitHub链接，有Star更好
4. **技术博客**：掘金/知乎/CSDN，展示学习能力

#### 投递技巧
- **分层投递**：
  - 第一层：大厂（练手，不抱希望）
  - 第二层：中厂（主要目标）
  - 第三层：小厂/外包（保底）
- **投递比例**：3:5:2（大厂:中厂:小厂）
- **投递时间**：周二-周四上午9-11点

#### 面试准备
- **自我介绍**：1-2分钟，突出项目经验
- **项目讲解**：用STAR法则，讲清楚技术难点
- **八股文**：JavaGuide至少过2遍
- **算法题**：LeetCode 200道（简单100+中等100）

#### 普通院校特别建议
- 学历是短板，用项目和技术弥补
- 先入行，工作2年后跳槽更容易
- 考虑二线城市，竞争压力小
- 外包公司可以作为跳板，不要长期待

## 路线B：AI应用开发工程师（新兴风口）

### B.1 岗位画像

**做什么**
AI应用开发工程师负责将大模型能力落地到具体业务场景，开发智能客服、知识库问答、AI助手等应用。这是2023-2024年最热门的新兴方向。

**适合谁**
- 对AI有热情，愿意持续学习新技术
- 有编程基础（Python或JavaScript）
- 想抓住AI风口，获得更高薪资
- 普通院校学生的弯道超车机会

**发展路径**
```
AI应用开发（1-2年）→ AI应用架构师（2-4年）→ AI技术负责人（4年+）
     15K-25K      →        30K-50K        →        50K-80K+
```

**普通院校策略**
- AI应用开发是新兴方向，学历要求相对宽松
- 项目经验比学历更重要
- 建议从传统开发转型，或直接切入
