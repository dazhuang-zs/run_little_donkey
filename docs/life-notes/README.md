# 人生笔记 — 抖音文字笔记库

> 程序员的碎片化创作系统，每条100-300字，金句感，可直接发抖音。

---

## 目录结构

```
life-notes/
├── README.md          ← 你正在看的文件
├── TEMPLATE.md        ← 笔记模板（必读）
│
├── 情绪共鸣/           ← 容易引发评论区互动的分类
│   ├── 01-焦虑时刻/
│   ├── 02-深夜独白/
│   ├── 03-治愈瞬间/
│   └── 04-中年危机/
│
├── 程序员视角/         ← 职业身份视角，技术圈的共鸣
│   ├── 01-代码人生/
│   ├── 02-职场真相/
│   ├── 03-技术悟道/
│   └── 04-AI时代/
│
├── 生活哲学/           ← 更普适的人生思考
│   ├── 01-时间管理/
│   ├── 02-人生选择/
│   ├── 03-自我和解/
│   └── 04-极简生活/
│
└── 灵感池/             ← 未分类的碎片想法，等待提炼
    └── drafts.md
```

---

## 分类说明

| 分类 | 定位 | 适合场景 | 热度预期 |
|-----|------|---------|---------|
| **情绪共鸣** | 情绪优先，故事载体 | 深夜、压力时刻、治愈内容 | ⭐⭐⭐⭐⭐ 最高 |
| **程序员视角** | 职业身份视角 | 技术圈共鸣、职场吐槽 | ⭐⭐⭐⭐ 高 |
| **生活哲学** | 普适人生思考 | 早上、周末、反思时刻 | ⭐⭐⭐ 中 |
| **灵感池** | 原始素材收集 | 随时记录，不急着发 | 内部用 |

---

## 文件命名规则

每个分类下的子文件夹内，笔记文件命名格式：

```
YYYY-MM-DD_关键词_情绪标签.md

示例：
2026-04-17_bug改完凌晨3点_焦虑.md
2026-04-17_35岁程序员_迷茫.md
2026-04-17_AI替不了我_自信.md
```

命名的好处：
- 时间排序自然
- 关键词一眼知道主题
- 情绪标签帮你选BGM

## 快速创作流程

1. **灵感捕捉** → 随手记到 `灵感池/drafts.md`
2. **情绪分类** → 移到对应分类文件夹
3. **润色打磨** → 加金句、调节奏
4. **BGM匹配** → 根据情绪标签选配乐
5. **发布复盘** → 记录数据，迭代模板

---

## 素材来源（程序员专属）

- 写代码时的突发感悟
- debug一整天后的崩溃瞬间
- 面试被问到不会的问题
- 看到新人代码想骂人的冲动
- AI工具带来的冲击和思考
- 加班回家路上的独处时间
- 技术社区的热点讨论
- 每周复盘时的自我对话

把这些瞬间记录下来，就是最好的素材。
