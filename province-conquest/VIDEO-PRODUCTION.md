# 问鼎中原 — 视频录制与后期指南

> 本文档指导如何将网页动画录制为成品视频。

---

## 录制前准备

### 硬件要求

| 设备 | 推荐 | 最低 |
|-----|------|------|
| 电脑 | Mac/Windows 高性能 | 能流畅运行浏览器即可 |
| 显示器 | 1920x1080+ | 1080P |
| 网络 | 稳定（加载地图数据） | - |
| 麦克风 | USB电容麦 | 普通麦克风 |

### 软件要求

| 软件 | 用途 | 推荐 |
|-----|------|------|
| OBS Studio | 屏幕录制 | ✅ 推荐 |
| ScreenFlow | Mac屏幕录制 | 可选 |
| 浏览器 | Chrome/Edge | Chrome |
| 音频软件 | Audacity | 录音处理 |
| 视频剪辑 | Premiere/FCP | 后期剪辑 |

### 浏览器设置

1. **全屏模式**：F11 或 Ctrl+Cmd+F
2. **开发者工具**：确保控制台无错误
3. **缩放**：100%（Ctrl+0 重置）
4. **书签栏**：隐藏或折叠

## 录制流程

### 方式一：OBS Studio（推荐）

#### 第一步：设置场景

```
1. 打开 OBS Studio
2. 点击「+」添加场景
3. 命名：「问鼎中原」
4. 添加来源：「窗口捕获」或「屏幕捕获」
```

#### 第二步：配置录制

```
设置 → 输出：
├─ 输出模式：高级
├─ 编码器：H.264 (NVENC/Apple Video Toolbox)
├─ 帧率：60fps（或30fps）
├─ 分辨率：1920x1080
└─ 质量：高质量

设置 → 音频：
├─ 采样率：48kHz
├─ 轨道1：系统音频（配乐）
└─ 轨道2：麦克风（旁白，可选）
```

#### 第三步：开始录制

```
1. 打开浏览器，进入问鼎中原
2. 全屏（F11）
3. 点击 OBS「开始录制」
4. 等待动画完成
5. 点击「停止录制」
```

### 方式二：浏览器开发者工具

```
1. 打开 Chrome 开发者工具（F12）
2. 按 Ctrl+Shift+P
3. 输入「Capture full size screenshot」
4. 仅限截图，不支持视频
```

## 动画控制

### 加速/减速

如需调整动画速度，修改 `js/animation.js`：

```javascript
// 找到动画延迟设置
const DELAY_BATTLE_SHOW = 2000;  // 对决展示延迟
const DELAY_CONQUEST = 1500;     // 征服动画时长
const DELAY_ROUND_PAUSE = 800;    // 轮间停顿
```

### 暂停录制

如需中途暂停：
- OBS 可随时暂停/继续
- 或使用快捷键 Ctrl+Shift+，暂停后继续

## 后期制作

### 导入素材

```
1. 打开 Premiere Pro / Final Cut Pro
2. 新建项目：1920x1080 24fps（或30fps）
3. 导入 OBS 录制的 MP4 文件
4. 导入配乐 MP3 文件
```

### 剪辑流程

| 步骤 | 内容 | 时长 |
|-----|------|------|
| 1 | 裁剪开头多余部分 | - |
| 2 | 对齐配乐 | - |
| 3 | 调整音量 | - |
| 4 | 添加字幕轨道 | - |
| 5 | 导出 MP4 | - |

### 配乐处理

```
1. 背景音乐音量为 30-40%
2. 人声旁白音量为 80-90%
3. 如有解说词单独轨道
```

### 字幕添加

| 类型 | 内容 | 工具 |
|-----|------|------|
| 开场字幕 | 问鼎中原 | Premiere/Edius |
| 对决字幕 | 省名 VS 省名 | AE/Pr |
| 结束字幕 | 铭记历史·爱我中华 | Pr/AE |

## 常见问题

### Q: 动画太慢/太快？

A: 修改 `js/animation.js` 中的延迟参数

### Q: 画面模糊？

A: 确保 OBS 分辨率设置为 1920x1080

### Q: 配乐和动画不同步？

A: 在剪辑软件中手动对齐，或调整动画启动时机

### Q: 导出后文件太大？

A: 降低码率至 15 Mbps 或压缩分辨率
